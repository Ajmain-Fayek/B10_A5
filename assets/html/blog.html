What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface for representing HTML, XML, and other structured documents as a tree structure. 1 It provides a way for JavaScript programs to interact with the content, structure, and style of a document.

How do you select an element from the DOM?

There are several ways to select an element from the DOM in JavaScript:

  • Using .getElementById()
  • Using .getElementsByClassName()
  • Using .getElementsByTagName()
  • Using .querySelector()
  • Using .querySelectorAll()
What is event delegation in the context of the DOM, and why is it useful?

Event Delegation in the DOM refers to the practice of attaching an event listener to a parent element rather than to each individual child element. This allows you to handle events for multiple child elements efficiently without having to add separate event listeners to each one.

Why is it useful?
Improved Performance: When dealing with a large number of child elements, adding event listeners to each one can significantly impact performance. Event delegation reduces the overhead by handling events at a higher level in the DOM tree.

Dynamic Content: If the number of child elements changes dynamically (e.g., through JavaScript or user interaction), event delegation ensures that new elements are automatically included without requiring additional code to attach event listeners.

Simplified Code: Event delegation can lead to cleaner and more concise code, as you only need to manage a single event listener instead of multiple ones.

How do you manipulate an element's attributes and styles using the DOM?

Manipulating Attributes
To change an element's attributes using JavaScript and the DOM, you can use the following methods:
1. getAttribute(): Retrieves the value of a specified attribute.
2. setAttribute(): Sets the value of a specified attribute.
3. removeAttribute(): Removes a specified attribute.


Manipulating Styles
To modify an element's styles using JavaScript and the DOM, you can use the following methods:
1. style.property: Directly sets the value of a specific CSS property.
2. classList: Manages the element's class list, which can be used to apply or remove CSS classes.
